2b. Option Chain Assembly + Greeks Computation:

Zerodha does NOT provide Greeks/IV. Build this:

from vollib.black_scholes.implied_volatility import implied_volatility
from vollib.black_scholes.greeks.analytical import delta, gamma, theta, vega

def compute_option_metrics(spot, strike, expiry_date, option_ltp, option_type, risk_free_rate=0.065):
    t = (expiry_date - today).days / 365.0
    flag = 'c' if option_type == 'CE' else 'p'
    iv = implied_volatility(option_ltp, spot, strike, t, risk_free_rate, flag)
    d = delta(flag, spot, strike, t, risk_free_rate, iv)
    g = gamma(flag, spot, strike, t, risk_free_rate, iv)
    th = theta(flag, spot, strike, t, risk_free_rate, iv)
    v = vega(flag, spot, strike, t, risk_free_rate, iv)
    return {'iv': iv, 'delta': d, 'gamma': g, 'theta': th, 'vega': v}
Risk-free rate: RBI 91-day T-bill yield (fetch from RBI website, ~6.5%)
For precision: derive from spot-futures basis: r = (F/S - 1) * (365/T)

def calculate_lot_size(account_capital, risk_pct, stop_distance_points, lot_size):

def compute_trailing_stop(entry_price, current_price, atr, market_strength, volume_spike):
    base_trail = 1.5 * atr

    # Volatility regime adaptation
    if india_vix < 15:
        trail_multiplier = 1.0   # Tight stop in low vol
    elif india_vix < 25:
        trail_multiplier = 1.5   # Normal
    else:
        trail_multiplier = 2.5   # Wide stop in high vol

    # Market strength tightening
    if market_strength > 70:
        trail_multiplier *= 0.85  # Tighter in strong trend

    # Volume spike tightening (institutional activity detected)
    if volume_spike:
        trail_multiplier *= 0.80  # Tighten on volume spike

    trail = base_trail * trail_multiplier
    stop = max(entry_price - trail, previous_stop)  # Trail only moves in profit direction
    return stop

The minimum mandatory pipeline on every PR is: ruff + mypy + bandit + pytest (80% branch coverage) + pip-audit + gitleaks.

The total cost is ~$6/month (Zerodha API only). Everything else is free/open-source. This is genuinely a zero-proprietary-dependency system.
